# Theme Factory 技能

此技能提供精选的专业字体和颜色主题集合，每个主题都有精心挑选的调色板和字体配对。一旦选择主题，就可以应用于任何 artifact。

## 目的

要将一致、专业的样式应用于演示幻灯片或其他 artifact，请使用此技能。每个主题包括：
- 带有十六进制代码的连贯调色板
- 用于标题和正文的互补字体配对
- 适合不同上下文和受众的独特视觉标识

## 使用说明

要将样式应用于幻灯片或其他 artifact：

1. **显示主题展示**：显示 `theme-showcase.pdf` 文件，让用户可以视觉上看到所有可用主题。不要对其进行任何修改；只需显示文件以供查看。
2. **询问他们的选择**：询问要将哪个主题应用于幻灯片
3. **等待选择**：获得关于所选主题的明确确认
4. **应用主题**：一旦选择了主题，将所选主题的颜色和字体应用于幻灯片/artifact

## 可用主题

以下 10 个主题可用，每个都在 `theme-showcase.pdf` 中展示：

1. **Ocean Depths** - 专业而平静的海洋主题
2. **Sunset Boulevard** - 温暖而充满活力的日落颜色
3. **Forest Canopy** - 自然而踏实的大地色调
4. **Modern Minimalist** - 干净而现代的灰度
5. **Golden Hour** - 丰富而温暖的秋季调色板
6. **Arctic Frost** - 凉爽而清脆的冬季灵感主题
7. **Desert Rose** - 柔和而精致的尘土色调
8. **Tech Innovation** - 大胆而现代的科技美学
9. **Botanical Garden** - 新鲜而有机的花园颜色
10. **Midnight Galaxy** - 戏剧性而宇宙般的深色调

## 主题详情

每个主题都在 `themes/` 目录中定义，包含完整的规格：
- 带有十六进制代码的连贯调色板
- 用于标题和正文的互补字体配对
- 适合不同上下文和受众的独特视觉标识

## 应用流程

选择首选主题后：
1. 从 `themes/` 目录读取相应的主题文件
2. 在整个幻灯片中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉标识

## 创建自己的主题
为了处理现有主题都不适用于 artifact 的情况，创建自定义主题。根据提供的输入，生成类似于上述的新主题。给主题一个类似的名称，描述字体/颜色组合代表什么。使用提供的基本描述选择适当的颜色/字体。生成主题后，显示以供审查和验证。之后，按上述方式应用主题。
