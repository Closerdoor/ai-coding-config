# OpenSpec 安装指南

OpenSpec 是一个规范驱动开发工具，用于定义和管理 API 规范、项目规范。

## 特点

- 规范驱动开发
- API 规范定义
- 项目规范管理
- 自动生成文档

## 全局安装

```bash
# 克隆到全局目录
cd ~/.claude/skills  # Claude Code
# 或
cd ~/.config/opencode/skills  # OpenCode

git clone https://github.com/Fission-AI/OpenSpec.git
```

### Windows 用户

```powershell
cd $env:USERPROFILE\.claude\skills
git clone https://github.com/Fission-AI/OpenSpec.git
```

## 使用方式

OpenSpec 帮助你：
- 定义项目规范
- 管理 API 规范
- 生成规范文档
- 验证代码符合规范
