# UI UX Pro Max 安装指南

UI/UX 设计智能工具，提供可搜索的设计数据库和设计系统生成。

## 特点

- 可搜索的设计数据库
- 设计系统生成
- UI/UX 指南
- 多框架支持（React、Vue、Flutter 等）

## 安装方式

### 方式一：项目级安装（推荐）

已包含在本仓库 `.opencode/skills/ui-ux-pro-max/` 目录中，复制即可使用。

### 方式二：全局安装

```bash
# npm 全局安装
npm install -g ui-ux-pro-max

# 或克隆到全局目录
cd ~/.claude/skills
git clone https://github.com/obra/ui-ux-pro-max.git
```

### Windows 用户

```powershell
npm install -g ui-ux-pro-max
```

## 使用方式

UI UX Pro Max 提供：
- 设计系统生成
- UI 组件指南
- UX 最佳实践
- 多框架设计模式

## 数据文件

工具包含多个数据文件：
- `colors.csv` - 颜色方案
- `typography.csv` - 字体排版
- `icons.csv` - 图标指南
- `styles.csv` - 样式模式
- `stacks/` - 各框架特定指南
