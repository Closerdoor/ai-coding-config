# Spec Kit 安装指南

Spec Kit 是 GitHub 官方的规范工具，用于项目规范和文档管理。

## 特点

- GitHub 官方工具
- 项目规范管理
- 文档自动化
- 与 GitHub 集成

## 全局安装

```bash
# 克隆到全局目录
cd ~/.claude/skills  # Claude Code
# 或
cd ~/.config/opencode/skills  # OpenCode

git clone https://github.com/github/spec-kit.git
```

### Windows 用户

```powershell
cd $env:USERPROFILE\.claude\skills
git clone https://github.com/github/spec-kit.git
```

## 使用方式

Spec Kit 帮助你：
- 创建项目规范
- 管理技术文档
- 与 GitHub 工作流集成
- 自动化文档更新
