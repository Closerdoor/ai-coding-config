# 项目名称

简短描述项目用途。

## 规则

**所有对话使用中文进行。**

**Git 提交信息必须使用中文。**

## 技术栈

- 前端：React + TypeScript + Tailwind CSS
- 后端：Node.js + Express
- 数据库：PostgreSQL

## 开发命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run test` - 运行测试
- `npm run lint` - 代码检查

## 目录结构

```
src/
├── components/    # React 组件
├── hooks/         # 自定义 Hooks
├── lib/           # 工具函数
├── pages/         # 页面组件
└── types/         # TypeScript 类型定义
```

## 注意事项

- 提交前请运行 `npm run lint` 和 `npm run test`
- 分支命名：feature/xxx, fix/xxx, chore/xxx
